Visual Studio Code

Para la creación de este portafolio, elegí Visual Studio Code como herramienta principal. Su interfaz intuitiva, combinada con funciones como autocompletado de código, vista previa en tiempo real y un ecosistema de extensiones, me permitió desarrollar con mayor eficiencia y precisión. Además, su sistema de organización de archivos y carpetas me ayudó a mantener una estructura clara entre los documentos HTML, estilos CSS y recursos visuales, optimizando mi flujo de trabajo.

HTML y CSS

Para desarrollar mi portafolio, primero organicé el contenido utilizando HTML, implementando diversas etiquetas para definir la estructura. Luego, apliqué estilos con CSS, experimentando con paletas de color, tipografías y otros para lograr un diseño coherente y visualmente atractivo. Este proceso me permitió combinar lógica con creatividad, consolidando mis habilidades.
